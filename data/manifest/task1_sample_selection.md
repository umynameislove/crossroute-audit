# Task 1 Sample Selection Table

Purpose: hand-selection table for Lane B / Phase 2 / Day 1 before writing `samples.jsonl`.
Images are grouped under `data/images/` by intended role. Source/license fields are included so Task 2 can keep the manifest reproducible.

## Folder Layout

- `data/images/high_visual_dependency/` — clean object/color-presence samples.
- `data/images/language_prior/` — samples answerable from language/world knowledge, used to exclude priors.
- `data/images/counterfactual/` — replacement images expected to flip selected clean samples.
- `data/images/negative_control/` — samples for later irrelevant-region masking/ablation, expected effect near zero.

- Total images: 30
